# ChatGpt

关于ChatGpt的使用的Guide，其实就是网络环境的限制导致某些地区的用户注册不便，本身是一个很简单的流程，总结2个操作 VPN+接收验证码

# 使用前提

  • 需要一个翻墙软件，并提供稳定的服务，在进行以下流程的时候请确保连接的节点为：
    韩国，日本，印度，新加坡，美国(特别提醒：香港节点不可用)
    个人推荐（新加坡节点） 延时低些，最终还是取决于你的节点质量，和线路繁忙与否
  • 建议打开你的VPN的全局代理模式
  • 选择Google Chrome浏览器 或者 Apple Safari浏览器
  • 支付宝可扫码支付1.2元人命币

# 第一步注册OpenAi账号

入口网址：OpenAI (opens new window)点进去之后点击Try ChatGpt

  • 先泽Sign注册一个账号（确保连接VPN）
  • 邮箱选择Gmail、网易163、QQ都OK
  • 在你输入密码的时候可能注册失败，大概率是网络环境问题（切换节点），如果你多次尝试注册都失败了，他会记住你的IP，你下次注册需要很长时间之后才能进行
  • 最近的用户量比较大，消息量也很大，如果多次尝试注册都失败了，也可能是他的规则改变了，你可以换个时间再注册

# 注册中的如何接受验证码问题

如果你进行到下一步，接下来需要输入
手机号，并填写验证码，这个时候我们需要进入这个网站
他会帮我们代理接受验证码并告诉我吗 前往地址 (opens new window)

  • 同样注册并登陆
  • 注册点击后，你需要去邮箱确认 去你的邮箱点击确认
  • 选择支付宝付款
  • 充值11 卢布左右就行，只接收一次短信
  • 然后去搜索OpenAI并点击选择印度就行
  • 然后复制号码
  • 然后你的sms-activate网页会给你反馈验证码，复制即可，如果失败了，可以换换成印尼、巴西、泰国之类的试试

# 登陆

确保你连的节点依然是美国、新加坡、韩国、日本（香港和裸连都是不可行的）
