1.把20号做的算法题，重新写了一个基于三项切分的快排和堆排序版本解题 215. 数组中的第K个最大元素 (opens new window)

//堆
class Solution {
    public int findKthLargest(int[] nums, int k) {
        PriorityQueue<Integer> pq = new PriorityQueue<>();
        for (int val : nums) {
            pq.add(val);
            if (pq.size() > k)
                pq.poll();
        }
        return pq.peek();
    }
}
//快速排序
class Solution {
    public int findKthLargest(int[] nums, int k) {
        k = nums.length - k;
        int i = 0 , j = nums.length - 1;
        while(i < j){
            int point = partition(nums, i, j);
            if(point == k){
                break;
            }else if(point < k){
                i = point + 1;
            }else{
                j = point - 1;
            }
        }
        return nums[k];
    }

    private int partition(int[] nums, int l, int r){
        int i = l , j = r + 1;
        int v = nums[l];
        while(true){
            while(nums[++i] < v)if(i == r)break;
            while(nums[--j] > v)if(j == l)break;
            if(i >= j)break;
            swap(nums, i, j);
        }
        swap(nums, l, j);
        return j ;
    }

    private void swap(int[] nums, int i, int j){
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }

}
//三项切分快速排序
class Solution {
    public int findKthLargest(int[] nums, int k) {
        k = nums.length - k;
        int i = 0 , j = nums.length - 1;
        while(i < j){
            //切分（三项切分，切出来的是三部分，< = >）
            partition(nums, i, j);
            if(k <= gt && k >= lt){
                break;
            }else if(k > gt){
                i = gt + 1;
            }else{ 
                j = lt - 1;
            }
        }
        return nums[k];
    }

    private int lt;
    private int gt;

    public void partition(int[] nums, int l, int r){
        lt = l;
        gt = r;
        int v = nums[l] , i = l + 1;
         //i的趋势是向右走，gt的趋势是向左走，i更靠近gt，i和gt的位置关系决定切分是否完成
         //然后是
        while(i <= gt){ 
            if(nums[i] < v){   //如何i位置元素小于v就交换给lt
                swap(nums, lt++, i++);
            }else if(nums[i] > v){   //如何i位置元素大于v就交换给gt
                swap(nums, gt--, i);
            }else i++; //相等的那部分
        }
    }
    
    private void swap(int[] nums, int i, int j){
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}

2.桶排的应用 LeetCode 347. 前 K 个高频元素 (opens new window)

class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        HashMap<Integer,Integer> frequencyForNum = new HashMap<>();
        for(int num : nums){
            frequencyForNum.put(num, frequencyForNum.getOrDefault(num, 0) + 1);
        }
        List<Integer>[] buckets = new ArrayList[nums.length + 1];
        frequencyForNum.forEach((key,frequency) -> {
            if (buckets[frequency] == null) {
                buckets[frequency] = new ArrayList<>();
            }
            buckets[frequency].add(key);
        });
        List<Integer> topK = new ArrayList<>();
        for (int i = buckets.length - 1; i >= 0 && topK.size() < k ; i--) {
            if(buckets[i] == null)continue;
            if(buckets[i].size() >= k){
                topK.addAll(buckets[i].subList(0, k - topK.size()));
            }else{
                topK.addAll(buckets[i]);
            }
        }
      	//用stream+lambda可以巧妙将List转换为int array～～～
        return topK.stream()
                .mapToInt(Integer::intValue)
                .toArray();
    }
}

451. 根据字符出现频率排序 (opens new window)

class Solution {
    public String frequencySort(String s) {
        byte[] bytes = s.getBytes();
        Arrays.sort(bytes);
        s = new String(bytes);
        //计算频数并记录
        HashMap<Character,Integer> frequencyMap = new HashMap<>();
        int n = s.length();
        for(int i = 0 ; i < n ; i ++){
            char ch = s.charAt(i);
            frequencyMap.put(ch,frequencyMap.getOrDefault(ch,0)+1);
        }
        //构造桶，将频数一致的元素放入同一个桶
        ArrayList<Character>[] buckets = new ArrayList[n + 1];
        for(int i = 0 ; i < n ; i ++){
            char ch = s.charAt(i);
            int frequency = frequencyMap.get(ch);
            if(buckets[frequency] == null)
                buckets[frequency] = new ArrayList<>();
            buckets[frequency].add(ch);
        }
        //倒序遍历，得到的就是频数从高到低的排列，因为我们对s进行过整理，不会出现顺序不一致的问题
        StringBuilder res = new StringBuilder();
        for(int i = n ; i >= 0 ; i --){
            ArrayList<Character> buket = buckets[i];
            if(buket == null)continue;
            buket.forEach(res::append);
        }
        return res.toString();
    }
}

class Solution {
    public void sortColors(int[] nums) {
        //明摆着要考你，快速排序的优化版本 三项切分版本，
        //这是我们大名鼎鼎Dijkstra提出的荷兰国旗问题，同样他给出了三项切分的解法
        //（两向切分）普通快速排序[小于v部分,大于v部分] QuickSort
        //三向切分快速排序[小于v部分,等于v部分,大于v部分] Quick3WaySort
        //如果你有<<算法>>这本书，可以翻到290的2.3.2，298的2.3.4 两个示意图
        quick3WaySort(nums, 0, nums.length - 1);
    }

    public void quick3WaySort(int[] nums, int l, int r){
        if(l >= r)return;
        int v = nums[l], lt = l, i = l + 1, gt = r;
        //三向切分 [<,=,>]
        while(i <= gt){
            //一定要注意xx++进入方法的后还是lt，方法结束完成之后才++，可以自己试一试
            if(nums[i] < v)
                swap(nums, lt ++, i++); 
            else if(nums[i] > v)
                swap(nums, gt -- , i);
            else i++;
        }
        //分治～～，归并排序也用到了分治
        quick3WaySort(nums, l, lt - 1);
        quick3WaySort(nums, gt + 1, r);
    }

    public void swap(int[] nums, int i, int j){
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
